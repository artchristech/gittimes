# Git Times — working context

AI-generated newspaper for builders, live at **gittimes.com** (GitHub Pages / gh-pages,
fronted by Cloudflare). Daily editions are generated on OpenRouter and published by CI.
On top sits the **human product (fully live, real money):** an interactive AI chat ("AI
Desk") on every edition, magic-link reader accounts, and **$5/mo Stripe premium on LIVE
keys** — all served by a Cloudflare Worker. There's also an **x402-paid agent API** on the
VPS (paywall OFF). As of **2026-06-23** the **AI Markets page** got a full visual overhaul +
interactivity (cost calculator, sortable/filterable board, compare, evals-honesty pass),
**shipped + deployed** (PR #1 squash → main `202b4de`, daily-edition deploy, live-verified).
**The active focus now is the AI Desk chat window** — critiqued in `docs/ai-desk-critique.md`
(Grade C-); next phase = the window redesign (dock left, ≥25% width, full height, fullscreen).

## How it runs
- **Daily edition:** `daily-edition.yml` → `npm test` → `node src/sync-models.js` (fresh
  prices+catalog) → `node publish-edition.js` (`getTickerData`/`getFullMarketData` →
  `renderMarketsPage` + writes `site/data/models.json`) → deploy `./site` to gh-pages.
- **Reader/login/premium:** the Worker. `/auth/send-magic-link` → Resend → `/auth/verify`
  → session (SESSIONS KV) → `/checkout` (Stripe LIVE) → webhook flips premium → `/chat`.
- **Chat tiering (`worker/index.js` ~/chat):** premium → monthly cap; **free registered →
  daily allowance**; anonymous → paywall. `buildSystemPrompt(market)` injects today's date +
  live model pricing fetched from `gittimes.com/data/models.json` (6h + in-memory cache).
- **Deploys (3 surfaces, all manual-ish):**
  1. **Worker** — authed `wrangler deploy` (account `mandalazenwave@gmail.com`). Pushing main does NOT deploy it.
  2. **Site, full rebuild** — `daily-edition.yml` (cron 07:00 UTC or `gh workflow run`). ⚠ This REGENERATES editorial content via the LLM AND **re-sends the newsletter** — do NOT use it to ship a UI/CSS/JS hotfix.
  3. **Site, surgical hotfix (use for chat/UI/CSS/markup)** — `git worktree add <tmp> origin/gh-pages` → transform the published HTML (or copy `chat.js`) → `git push origin HEAD:gh-pages`. No content regen, no newsletter. Editorial content is NOT stored re-renderably (`editions/history.json` = star snapshots only), so you cannot re-render pages from data — you transform the built HTML or full-rebuild.
  - **Pages quirk:** first push often returns a transient `Page build failed` (duration:0); re-request with `gh api -X POST repos/artchristech/gittimes/pages/builds`, then poll `…/pages/builds/latest`.
  - **Cloudflare** caches HTML `max-age=600` and **ignores `?query` cache-busters**; force a fresh read with `curl -H "Cache-Control: no-cache"`, else wait ~10 min for the edge TTL.

   price `price_1TlSlFRDGztz1bRj4ppyPdCA` + live webhook `we_1TlSlNRDGztz1bRj8vHCpQJ6`.

2. **The markets "outdated" bug was `getFullMarketData()` returning null on a healthy sync** —
   the full catalog only existed in a live-fetch cache, so "All Models by Provider" silently
   vanished. Fixed: sync-models persists `catalog`; ai-ticker reads it. Don't reintroduce a
   live-only catalog path.
3. **Worker deploys via wrangler OAuth, not a token.** OAuth reads the CF zone but can't write
   DNS; DNS needs the `cfut_…` token. `deploy-worker.yml` skips gracefully unless `CF_API_TOKEN` set.
4. **KV CLI defaults to LOCAL** — `wrangler kv key …` needs `--remote` to touch deployed KV.
5. **Email verified** — `gittimes.com` in Resend (DKIM+SPF in Cloudflare), `EMAIL_FROM=noreply@gittimes.com`.
6. **Cloudflare zone** `gittimes.com` = `6621de9204d4b729451b83523c509c78`. Edge-caches the site;
   `?query` busters DON'T bypass it — use `curl -H "Cache-Control: no-cache"` or wait ~10 min.
7. **CSP must include `'self'` in `script-src`** (fixed 2026-06-23 in all `templates/*.html`). It
   had only `'unsafe-inline'`, which silently blocked the external `<script src="/chat.js">` so
   the chat fab's listener never attached — inline scripts ran, externals didn't. Any NEW
   external/bundled JS needs `'self'`; `{{CSP_SCRIPT_SRC}}` (`src/template-utils.js`) only adds analytics origins.
8. **Chat UI = 3 files, no build step.** `src/render.js` (`renderChatUi`/`renderHybridArticle`/
   `renderQuickHit`), `public/chat.js` (plain ES5 `var`, copied to site root by `publish.js` step 15;
   NOT in the eslint glob — hand-check it), `styles/newspaper.css` (inlined into every page at build,
   so a CSS change needs a re-render or an inlined-style transform to go live).
